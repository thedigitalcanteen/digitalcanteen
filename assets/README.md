# The Digital Canteen - Static Website

A fully static landing page for The Digital Canteen school meal management platform.

## 🚀 Quick Start

Simply open `index.html` in any modern web browser. No build tools, npm, or server required!

```bash
# Option 1: Double-click index.html to open in your default browser

# Option 2: Use a simple HTTP server (optional)
python -m http.server 8000
# Then visit http://localhost:8000
```

## ✨ Features

- ✅ **No Dependencies** - Pure HTML, CSS, and JavaScript
- ✅ **Fully Responsive** - Mobile, tablet, and desktop optimized
- ✅ **Interactive Carousel** - Touch/swipe support for mobile
- ✅ **Request Demo Modal** - Form validation and confirmation
- ✅ **Smooth Scrolling** - Navigation with smooth scroll
- ✅ **Custom Icons** - Lucide icons via CDN
- ✅ **Custom Fonts** - Google Fonts (Manrope & Outfit)

## 🎨 Sections

1. **Hero Section** - Main call-to-action with branding
2. **About Us** - Company information and key features
3. **Website Showcase** - Dashboard icons and screenshots
4. **Dashboard Carousel** - Interactive preview of all dashboards
5. **Features** - 6 key platform features
6. **Vision & Mission** - Company values and goals
7. **Footer** - Contact info, links, and social media

## 🔧 External Resources (CDN)

The website uses the following external resources:

- **Fonts**: Google Fonts (Manrope, Outfit)

## 🎯 Customization

### Colors

The color scheme is defined in CSS variables in `style.css`:

```css
:root {
    --yellow-400: #FACC15;
    --black: #000000;
    --white: #FFFFFF;
    /* ... more colors */
}
```

### Images

Replace images in `assets/images/` folder with your own. Keep the same filenames or update references in `index.html`.

### Content

Edit text directly in `index.html`. The structure is semantic and easy to navigate.
