# The Digital Canteen static demo

This folder is a static HTML export of the PHP prototype. It is intended for GitHub Pages.

Start here:
- index.html for the dashboard selector
- login.html for the login/sign-up page

Notes:
- PHP includes have been rendered into HTML.
- Links were changed from .php to .html.
- Dynamic PHP query-based variations are not fully preserved; this is suitable as a clickable prototype/demo, not a real backend system.
- You can open index.html directly in your browser, or host the folder in GitHub Pages.
